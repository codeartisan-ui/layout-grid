Responsive Grid

A responsive grid is very essential when you are developing mobile first web. It has 12, 8 and 4 columns for the desktop, tablet and mobile view respectively. All the grid size and breakpoints follow the material design guidelines precisely. However you can make your own breakpoints by editing Sass variables.
